This main.py has python code to create panorama with three input images using affine and homographic transformations.

##print("Usage: [Question_Number] [Input_Options] [Output_Options]")
To do perspective warping the command that should be used is - 
python main.py 1 <input1.png> <input2.png> <input3.png> <./Results> 

To do Cylindrical warping the command that should be used is - 
python main.py 2 <input1.png> <input2.png> <input3.png> <./Results> 


To do perspective warping with image blending the command that should be used is - 
python main.py 3 <input1.png> <input2.png> <input3.png> <./Results> 

To do Cylindrical warping with image blending the command that should be used is - 
python main.py 4 <input1.png> <input2.png> <input3.png> <./Results>
